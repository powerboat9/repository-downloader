#Syntax

|Feature|Meaning|
|:----:|:----:|
|{}|Repeat 1 or more times|
|[]|Optional|
|()|User specified value|

#Command Arguments

download {package}

Append "!" to the start of a package name to disable the removal of .lua file extensions

# Package Format

(user)/(repo)[:(branch)][@(gitPath)][>(savePath)]
